# Currency Button Component
The Currency Button component is a reusable component designed for React Native applications. It provides a button interface with a flag and the name of a currency. This component is useful for creating currency selection buttons within an application.

## Learnings
Through the development of this component, I gained experience in:
- Building reusable components in React Native
- Working with props to customize component behavior
- Utilizing StyleSheet for styling components
- Enhancing the user interface with emojis and text

## Key Takeaways
- Understanding of component composition and reusability
- Improved proficiency in React Native development
- Importance of clean and modular code structure
- Enhanced UI design skills with emoji integration
